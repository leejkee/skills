# Skills Repo

## Image-optimizer
用于opencode中，让llm自动处理windows截图，将处理结果放到Blog的静态资源目录

- Windows默认截图目录为：`~/Pictures/Screenshots/`

- 如果你使用`Linux/Macos`，应该自行修改`process_img.py`的`input`参数，或者直接让llm传入图片路径
